## Brand

- **Concept:** Luxury Editorial, Minimal Fashion, Museum Architecture, High Fashion Gallery
- **Aesthetic:** Clean, massive white volumes, concrete, black steel, mirror, editorial lighting, negative space
- **Experience:** Walking inside an editorial photoshoot

## Typography

- **Display:** Playfair Display / Canela
- **Body:** Inter / Neue Montreal
- **Mono:** SF Mono / Fira Code

## Stack

- Vite + React + TypeScript
- SCSS with design tokens
- GSAP + ScrollTrigger
- Lenis smooth scroll
- SplitType text splitting
- Barba.js page transitions

## Sections

1. **Loader** вЂ” Editorial loader with white canvas wipes, large typography, film grain
2. **Navigation** вЂ” Minimal top bar with mobile menu
3. **Hero** вЂ” Fullscreen editorial with campaign film, huge typography, mouse-reactive camera
4. **Manifesto** вЂ” "We don't serve coffee. We compose moments." pinned section
5. **Collection** вЂ” Drinks as fashion collections (Collection 01 Black Espresso, etc.)
6. **Editorial Gallery** вЂ” 15 fashion campaign photographs with magazine spacing
7. **Runway Film** вЂ” Autoplay silent video, pinned cinematic section
8. **Creative Process** вЂ” Grid with storyboards, material samples, sketches
9. **The Studio** вЂ” Concrete, steel, white interiors with parallax
10. **Fashion Calendar** вЂ” Upcoming collaborations and launches
11. **Journal** вЂ” Magazine articles with huge images
12. **Footer** вЂ” Animated logo, infinite marquee, credits
